PyFAI Plugins
=============

PyFAI is a python library for performing azimuthal regrouping.
This plugin directory contains plugin accapted by various applications or frameworks.

LImA
----
Library for Image Acquisition is a library written by the Bliss people at ESRF (http://lima.blissgarden.org/)
The aim of this plugin is to create a "LinkTask" in their processLib which is able to perform an azimuthal regrouping inside the software controling the camera.


EDNA
----
EDNA is a framework for plugin based application.
A generic pyFAI plugin is planed soon.


PyMca
-----
PyMca is an X-ray fluorescence and map analysis tool.
This pyFAI plugin will provide azimuthal regrouping capabilities to PyMca.
